## ⚠️ 已知限制与改进空间

### 当前限制
1. **Jina AI 403错误** - 上游API限制，已通过多源架构缓解
2. **DuckDuckGo 解析** - 依赖HTML结构，可能因网页变更失效
3. **LLM 依赖** - 需要外部 API Key，无 Key 时使用简单摘要

### LLM 配置说明
- 配置环境变量即可启用：`INFO_GATHERER_LLM_API_KEY`
- 支持自定义 base_url（OpenRouter、本地模型等）
- 超时可配置，失败自动回退不影响整体流程

### 后续计划（README.md 中已规划）
